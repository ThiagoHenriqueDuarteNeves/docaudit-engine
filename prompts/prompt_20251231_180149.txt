## Selecting a UI Mode
You can configure LM Studio to run in increasing levels of configurability.
Select between User, Power User, and Developer at the bottom of the app.
## Which mode should I choose?
### User
Show only the chat interface, and auto-configure everything. This is the best 
choice for beginners or anyone who's happy with the default settings.
### Power User
Use LM Studio in this mode if you want access to configurable load and inference
parameters as well as advanced chat features such as insert, edit, & continue 
(for either role, user or assistant).
### Developer
Full access to all aspects in LM Studio. This includes keyboard shortcuts and 
development features. Check out the Developer section under Settings for more.
